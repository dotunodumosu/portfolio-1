Portfolio

Eventify

React application allowing to search for concerts of a specific artist or every concert in the exact city. Users can also buy a ticket for the chosen concert (application redirect to a seller website). Logged in users can also save favourite artists and concerts. Users authorization and managing data was created in Firebase.

  • React
  • Firebase

Yahtzee | Dice Game

My version of popular dice game - Yahtzee. It's the first version of the game, where one player can play a single round. This application was written in React. I used styled-components for styling and Webpack as a bundler. The application allows saving the final score in a Firebase database.

  • React
  • Firebase

Coffee Republic

An example of a fictional cafe website. I designed and created it by myself using GatsbyJS and GraphQL. The individual posts pages are generated automatically from markdown files.

About me

My name is Paweł and I am a Front-end Developer from Gdańsk. I graduated in Economics at WSB University in Gdynia. I have been working in a bank for a few years before I decided to learn web development. I have included more details about my professional career in my CV (download link below) and my Linkedin profile.

I use correct and semantic HMTL and CSS code in my projects, with SCSS preprocessor. I'm trying to keep up with current standards in my JavaScript code to make simple interactions and more complex actions on my websites. I'm strongly focused on learning React currently. I use Figma to create designs of my projects, I'm also familiar with Adobe Photoshop and Adobe XD. Most of the time I bundled my projects with Webpack (just a simple configuration for a small project). I can also use GatsbyJS and GraphQL to build a static website. I used Firebase for authorization and managing data in my recent projects.

I speak English freely (spoken and written). I have FCE and LCCI 3 certificates.

Check out my Github repository for more details about my projects.
